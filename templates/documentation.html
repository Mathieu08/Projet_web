Documentation de l'API


GET
http://localhost:5000/api/articles

Retourne une liste sous format JSON contenant tous les articles publiés et le serveur renvoie un code 200.
Les objets articles de la liste JSON contiennent le titre, l'auteur et l'url complète pour chaque article.


GET
http://localhost:5000/api/article/{identifiant}

Retourne les informations de l'article correspondant à l'identifiant passé en paramètre et le serveur renvoie un code 200.
Si l'identifiant passé en paramètre est inconnu le serveur renvoie un code 404.


POST
http://localhost:5000/api/creer-article

Créer un nouvel article à partir des données fournies sous format JSON.
Pour utiliser ce service il faut être authentifié.
Si l'objet est créé avec succès le serveur renvoie un code 201, si la requête est invalide ou l'identifiant existe déjà le serveur renvoie un code 400.
