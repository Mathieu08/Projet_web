Pour avoir accès à la partie admin il faut entrer comme nom d'utilisateur: correcteur
et comme mot de passe: secret .

Pour pouvoir utiliser les envoie de courriel il faut remplacer les données du fichier "email.json"
en spécifiant l'adresse courriel à utiliser ainsi que le mot de passe de celle-ci.

La documentation sur les services fournis par l'API se trouve à l'url: http://localhost:5000/api/documentation
